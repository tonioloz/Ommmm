# Ommmm

Ommmm es un generador de dibujos a partir de sonidos. Activa el micrófono y observa cómo el lienzo reacciona en tiempo real a la intensidad del audio.

## Qué hace

Una experiencia visual audio-reactiva en el navegador: el sonido se transforma en trazos que evolucionan con la energía de tu voz o del ambiente.

## Features

- Dibujo reactivo en tiempo real al nivel de audio
- Controles simples para iniciar/detener el micrófono
- Interfaz minimalista centrada en la experiencia
- 100% client-side

## Cómo usarlo

1. Abre la app en el navegador.
2. Toca "Start Mic".
3. Habla o reproduce sonido y observa el dibujo reaccionar.

## Stack

- Vite
- React
- Tailwind CSS
- Motion

## Desarrollo local

1. Instala dependencias:
   ```bash
   npm install
   ```
2. Arranca el servidor:
   ```bash
   npm run dev
   ```
3. Abre la URL que aparece en la terminal.

## Deploy en Vercel

1. Compila la app:
   ```bash
   npm run build
   ```
2. Sube el `dist/` a Vercel (preset Vite, sin configuración extra).
